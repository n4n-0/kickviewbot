# Kick View Bot

This application is a simple bot control UI built with React.js. It offers an interface to control and configure a bot that interacts with `kick.com`.

![preview](https://i.imgur.com/oysnEMo.png)

## Installation

Before running the application, make sure you have [Node.js](https://nodejs.org/) installed.

1. Clone the repository:
    ```
    git clone https://github.com/yourusername/kick-view-bot.git
    ```
2. Navigate to the project directory:
    ```
    cd kick-view-bot
    ```
3. Install dependencies:
    ```
    npm install
    ```

## Running the App

To start the application, run the following command in the project directory:

`npm start`

The application will be accessible at `http://localhost:3000`.

## Features

- **URL Configuration**: Enter a URL for the bot to interact with. The URL must contain "kick.com".
- **Number of Browsers**: Configure the number of browser instances that the bot will operate on.
- **Headless Mode**: Toggle headless mode for the browsers.
- **Start/Stop Bot**: Start or stop the bot operation. During the start process, the button will be disabled and display "Starting...". Once the bot is running, the button turns red and display "Stop Bot".
